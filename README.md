🎓 KENZO AI Study Assistant

An AI-powered educational platform that helps students learn smarter by generating quizzes, answering questions from documents, creating question papers, tracking study tasks, and analyzing learning progress.

---

✨ Features

🤖 AI Tutor

- Ask questions on any topic
- Get AI-generated explanations
- Supports conceptual learning and doubt solving
- Activity tracking in analytics

📄 PDF Assistant

- Upload PDF notes or previous year papers
- Ask questions from uploaded documents
- Get instant answers from document content

🧠 Quiz Generator

- Generate quizzes on any topic
- Select difficulty level (Easy, Medium, Hard)
- Multi-language support
- Automatic scoring and analytics tracking

📝 Question Paper Generator

- Generate custom exam papers
- Enter subject and chapters/topics
- Define marks pattern and difficulty level
- Generate answer keys
- Supports custom instructions

✅ Task Manager

- Create and manage study tasks
- Mark tasks as completed
- Track study progress

📊 Analytics Dashboard

- AI Tutor questions count
- Quiz attempts and scores
- PDF uploads count
- Papers generated
- Completed tasks
- Recent activity tracking
- Most-used feature statistics

---

🛠️ Technology Stack

- Frontend: Streamlit
- Backend: Python
- AI Model: Google Gemini API
- PDF Processing: PyPDF2 / PDFPlumber
- State Management: Streamlit Session State
- Version Control: Git & GitHub

🚀 How to Run

Clone Repository

git clone <repository-url>
cd <project-folder>

Install Dependencies

pip install -r requirements.txt

Run Application

python -m streamlit run app.py
